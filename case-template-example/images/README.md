# Изображения проекта

Файлы в этой папке — заменяемые примеры из существующего кейса Japanhouse. Перед передачей нового проекта замените их своими изображениями.

## Имена и назначение

| Файл | Назначение | Рекомендуемый размер |
|---|---|---:|
| `cover.webp` | Обложка кейса и карточки | 2400×1350, 16:9 |
| `01-overview.webp` | Общий вид результата | 2000×1500, 4:3 |
| `02-system.webp` | Система или основные варианты | 2000×1500, 4:3 |
| `03-detail.webp` | Важная деталь | 2000×1500, 4:3 |
| `04-application.webp` | Применение на носителе | 2000×1500, 4:3 |
| `05-result.webp` | Финальный результат в контексте | 2000×1500, 4:3 |

Можно оставить от одного до восьми изображений галереи. Неиспользуемые файлы удалите, а дополнительные называйте `06-name.webp`, `07-name.webp` и так далее.

## Экспорт

- Формат: WebP.
- Профиль: sRGB.
- Обложка: желательно до 400 КБ.
- Галерея: желательно до 600 КБ на файл.
- Не размещайте поверх изображения подписи интерфейса — сайт выводит их отдельно.
